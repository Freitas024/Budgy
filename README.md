# 📝 Budgy – Sistema de Solicitação e Gerenciamento de Orçamentos

> **Status do Projeto:** Em desenvolvimento (MVP) 🚀

O **Budgy** é uma solução SaaS multi-tenant desenvolvida para simplificar e profissionalizar o processo de orçamentos para pequenas empresas. O sistema permite que administradores gerenciem pedidos e organizem clientes através de um painel inteligente e intuitivo.

## ✨ Diferenciais Técnicos

Este projeto não é apenas uma interface; ele foi planejado com decisões de engenharia robustas:

* **Arquitetura Multi-tenant:** Estrutura preparada para que múltiplas empresas utilizem o software com total isolamento e segurança de dados.
* **Design-to-Code:** Fidelidade total entre o protótipo desenvolvido no Figma e a implementação final em React.
* **Foco em BI:** Dashboards pensados para entregar KPIs financeiros críticos para o pequeno empreendedor.
